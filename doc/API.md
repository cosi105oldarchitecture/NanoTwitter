# Overview

**Tweets**

* `GET` `/`
* `GET` `/tweets/:user_id`
* `GET` `/tweets/mentions`
* `POST` `/tweets/new` 

**Users**

* `GET` `/users/followers`
* `POST` `/users/followers`
* `GET` `/users/following`
* `POST` `/users/following`

**Log-In**

* `GET` `/login`
* `POST` `/login`

**Sign-Up**

* `GET` `/signup`
* `POST` `/signup`
